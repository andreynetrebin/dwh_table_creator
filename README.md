# DWH Table Creator
<img width="890" height="449" alt="dwh_table_creator " src="https://github.com/user-attachments/assets/b515e9e5-b041-4153-a466-61829dc4483f" />

DWH Table Creator — это веб-приложение, предназначенное для быстрого создания таблиц в Greenplum из CSV файлов с
использованием PXF и HDFS в качестве источника данных.

## Функциональные возможности

- Развернут Flask сервер для обработки запросов.
- Форма для загрузки CSV файлов.
- Обработка заголовков CSV файлов.
- Маппинг заголовков к именам столбцов с указанием типов данных и наименования таблицы.
- Загрузка CSV файлов в директорию на HDFS.
- Создание внешних таблиц из CSV файлов с использованием PXF.
- Создание таблиц на основе внешних таблиц.
- Назначение ролей и пользователей на созданные таблицы.

## Стек технологий

- **Python**: Язык программирования, используемый для разработки приложения.
- **Flask**: Веб-фреймворк для разработки серверной части приложения.
- **HDFS**: Система распределенного хранения данных, используемая для хранения загруженных CSV файлов.
- **Greenplum**: Система управления базами данных, в которую загружаются данные.
- **PXF (Platform Extension Framework)**: Используется для интеграции с HDFS и создания внешних таблиц.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/andreynetrebin/dwh_table_creator.git
   cd dwh_table_creator
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Настройте параметры подключения к Greenplum и в конфигурационном файле.
4. Запустите Flask сервер:
   ```bash
   python app.py
   ``` 

## Использование

1. Откройте веб-браузер и перейдите по адресу http://localhost:5000.
2. Загрузите CSV файл через форму.
3. Укажите необходимые параметры, такие как наименование таблицы и типы данных.
4. Нажмите кнопку для создания таблицы.
5. Задайте роли и пользователей для созданной таблицы.
